## 2.0.0（2026-06-04）
- 重要升级 媒体库整体 UI 升级为侧边栏工作台布局，统一媒体资源、上传、免费素材库、标签管理等入口
- 新增 扩展存储支持，目前支持七牛云扩展存储，上传、删除、预览等流程会根据存储来源处理资源 [文档](#ext-storage)
- 新增 标签管理能力，支持创建标签、按标签筛选媒体、上传时关联标签、为单个或批量媒体设置标签
- 新增 独立上传页，支持选择存储位置、按图片/视频类型上传，并支持点击选择、拖拽、粘贴上传
- 新增 图片裁剪功能，可在媒体详情中对图片进行裁剪并上传保存裁剪结果
- 优化 从免费素材库选择资源后加入上传列表，可再选择存储位置和标签后保存到媒体库
- 优化 媒体信息面板，支持资源信息展示、标签维护、图片裁剪上传等能力
- 优化 媒体列表、免费素材库列表和筛选操作的交互体验
